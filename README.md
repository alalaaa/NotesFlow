# NotesFlow

Aplikacja do zarządzania notatkami i zadaniami z podziałem na strony.

## Technologie
- Python / Flask
- HTML, CSS, JavaScript
- SQLite (planowane)

## Funkcjonalności
- Tworzenie i zarządzanie notatkami oraz zadaniami
- Podział treści na strony
- System tagów
- Drag & drop
- Przypomnienia
- Tryb ciemny / jasny

## Uruchomienie
1. Sklonuj repozytorium: `git clone ...`
2. Zainstaluj zależności: `pip install -r requirements.txt`
3. Uruchom aplikację: `python app.py`
4. Otwórz w przeglądarce: `http://localhost:5000`
